# Backend for thinkpad store (just a class project)

## Setup
```shell
pip install -r requirements.txt
```

Then you can run:

```shell
python manage.py makemigration
python manage.py migrate
python manage.py runserver
```

> `runserver` can be given a argument as address, defaults to `127.0.0.1:8000`

## doc
Ref server's `/swagger/` for api documents after running servers
